# 帮助文档管理系统用户手册

## 1. 系统概述

帮助文档管理系统是一个功能全面的文档管理平台，支持文档的创建、编辑、版本控制、分类管理、搜索查询和权限控制等核心功能。系统采用现代化的用户界面设计，提供直观的操作体验，适用于企业内部文档管理、技术文档发布、用户指南编写等场景。

## 2. 系统功能

### 2.1 文档管理
- **文档创建**：支持创建Markdown、HTML和纯文本格式的文档
- **文档编辑**：提供富文本编辑器和Markdown编辑器，支持实时预览
- **文档发布**：可以将文档发布为公开、私有或受保护状态
- **文档归档**：可以将不再使用的文档归档
- **文档删除**：可以删除不需要的文档

### 2.2 版本控制
- **版本历史**：自动记录文档的所有修改版本
- **版本比较**：支持比较不同版本之间的差异
- **版本恢复**：可以恢复到之前的任何版本
- **版本下载**：可以下载特定版本的文档

### 2.3 分类管理
- **分类树**：支持多级分类结构
- **分类编辑**：可以创建、编辑和删除分类
- **分类排序**：可以调整分类的显示顺序
- **分类统计**：显示每个分类下的文档数量

### 2.4 搜索查询
- **全文搜索**：支持对文档标题和内容进行全文搜索
- **分类筛选**：可以按分类筛选文档
- **状态筛选**：可以按文档状态（发布、草稿、归档）筛选
- **标签筛选**：可以按标签筛选文档

### 2.5 权限管理
- **角色管理**：支持管理员、编辑和查看者等角色
- **权限分配**：可以为不同角色分配不同的操作权限
- **文档权限**：可以为单个文档设置访问权限
- **用户管理**：可以管理系统用户和分配角色

### 2.6 数据备份与恢复
- **手动备份**：可以手动创建系统备份
- **自动备份**：支持定时自动备份
- **备份恢复**：可以从备份文件恢复系统数据
- **备份管理**：可以管理备份文件和查看备份状态

## 3. 系统架构

### 3.1 技术栈
- **前端框架**：Vue 3 + TypeScript
- **UI组件库**：Arco Design Vue
- **状态管理**：Pinia
- **路由管理**：Vue Router 4
- **HTTP客户端**：Axios
- **富文本编辑器**：Tiptap
- **Markdown渲染**：Marked
- **代码高亮**：Highlight.js
- **Mock数据**：Mock.js

### 3.2 系统结构
```
├── src/
│   ├── api/              # API接口定义
│   ├── components/       # 公共组件
│   ├── mock/             # Mock数据
│   ├── router/           # 路由配置
│   ├── store/            # 状态管理
│   ├── views/            # 页面组件
│   │   └── help-doc/     # 帮助文档管理系统页面
│   ├── App.vue           # 根组件
│   └── main.ts           # 入口文件
├── docs/                 # 文档目录
└── package.json          # 项目配置
```

## 5. 使用指南

### 5.1 登录系统
1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击登录按钮

### 5.2 文档管理

#### 5.2.1 创建文档
1. 点击左侧导航栏中的"文档管理"->"文档列表"
2. 点击"新建文档"按钮
3. 填写文档标题、内容、分类、标签等信息
4. 点击"保存"按钮保存为草稿，或点击"发布"按钮发布文档

#### 5.2.2 编辑文档
1. 在文档列表中找到需要编辑的文档
2. 点击文档右侧的"编辑"按钮
3. 修改文档内容
4. 点击"保存"按钮保存修改

#### 5.2.3 发布文档
1. 在文档列表中找到需要发布的文档
2. 点击文档右侧的"发布"按钮
3. 确认发布操作

#### 5.2.4 归档文档
1. 在文档列表中找到需要归档的文档
2. 点击文档右侧的"归档"按钮
3. 确认归档操作

#### 5.2.5 删除文档
1. 在文档列表中找到需要删除的文档
2. 点击文档右侧的"删除"按钮
3. 确认删除操作

### 5.3 版本管理

#### 5.3.1 查看版本历史
1. 在文档列表中找到需要查看版本历史的文档
2. 点击文档右侧的"版本历史"按钮
3. 在版本历史页面中查看所有版本

#### 5.3.2 比较版本差异
1. 在版本历史页面中选择两个版本
2. 点击"比较版本"按钮
3. 查看版本之间的差异

#### 5.3.3 恢复版本
1. 在版本历史页面中找到需要恢复的版本
2. 点击版本右侧的"恢复"按钮
3. 确认恢复操作

### 5.4 分类管理

#### 5.4.1 创建分类
1. 点击左侧导航栏中的"文档管理"->"分类管理"
2. 点击"添加分类"按钮
3. 填写分类名称、描述等信息
4. 选择父分类（可选）
5. 点击"保存"按钮

#### 5.4.2 编辑分类
1. 在分类树中找到需要编辑的分类
2. 点击分类右侧的"编辑"按钮
3. 修改分类信息
4. 点击"保存"按钮

#### 5.4.3 删除分类
1. 在分类树中找到需要删除的分类
2. 点击分类右侧的"删除"按钮
3. 确认删除操作

#### 5.4.4 调整分类顺序
1. 在分类树中找到需要调整顺序的分类
2. 点击分类右侧的"上移"或"下移"按钮
3. 调整分类的显示顺序

### 5.5 权限管理

#### 5.5.1 角色管理
1. 点击左侧导航栏中的"系统管理"->"角色管理"
2. 可以查看、创建、编辑和删除角色
3. 为每个角色分配不同的权限

#### 5.5.2 用户管理
1. 点击左侧导航栏中的"系统管理"->"用户管理"
2. 可以查看、创建、编辑和删除用户
3. 为每个用户分配角色

### 5.6 数据备份与恢复

#### 5.6.1 手动备份
1. 点击左侧导航栏中的"系统管理"->"备份与恢复"
2. 切换到"备份"标签页
3. 选择备份类型
4. 填写备份描述
5. 点击"创建备份"按钮

#### 5.6.2 自动备份设置
1. 点击左侧导航栏中的"系统管理"->"备份与恢复"
2. 切换到"设置"标签页
3. 启用自动备份
4. 设置备份间隔和时间
5. 设置备份保留天数
6. 点击"保存设置"按钮

#### 5.6.3 恢复数据
1. 点击左侧导航栏中的"系统管理"->"备份与恢复"
2. 切换到"恢复"标签页
3. 选择需要恢复的备份文件
4. 点击"恢复"按钮
5. 确认恢复操作

## 6. API接口

### 6.1 文档接口
- `GET /api/help-doc/documents` - 获取文档列表
- `GET /api/help-doc/documents/:id` - 获取单个文档
- `POST /api/help-doc/documents` - 创建文档
- `PUT /api/help-doc/documents/:id` - 更新文档
- `DELETE /api/help-doc/documents/:id` - 删除文档
- `POST /api/help-doc/documents/batch-delete` - 批量删除文档
- `POST /api/help-doc/documents/batch-publish` - 批量发布文档
- `POST /api/help-doc/documents/batch-archive` - 批量归档文档
- `POST /api/help-doc/documents/export` - 导出文档
- `POST /api/help-doc/documents/import` - 导入文档

### 6.2 版本接口
- `GET /api/help-doc/versions` - 获取版本列表
- `GET /api/help-doc/versions/:id` - 获取单个版本
- `POST /api/help-doc/versions/compare` - 比较版本差异
- `POST /api/help-doc/versions/restore` - 恢复版本
- `POST /api/help-doc/versions/:id/download` - 下载版本

### 6.3 分类接口
- `GET /api/help-doc/categories` - 获取分类列表
- `GET /api/help-doc/categories/:id` - 获取单个分类
- `POST /api/help-doc/categories` - 创建分类
- `PUT /api/help-doc/categories/:id` - 更新分类
- `DELETE /api/help-doc/categories/:id` - 删除分类
- `POST /api/help-doc/categories/sort` - 调整分类顺序

### 6.4 权限接口
- `GET /api/help-doc/roles` - 获取角色列表
- `GET /api/help-doc/roles/:id` - 获取单个角色
- `POST /api/help-doc/roles` - 创建角色
- `PUT /api/help-doc/roles/:id` - 更新角色
- `DELETE /api/help-doc/roles/:id` - 删除角色
- `POST /api/help-doc/roles/:id/permissions` - 分配角色权限

### 6.5 备份接口
- `GET /api/help-doc/backups` - 获取备份列表
- `POST /api/help-doc/backups` - 创建备份
- `DELETE /api/help-doc/backups/:id` - 删除备份
- `POST /api/help-doc/backups/:id/download` - 下载备份

### 6.6 恢复接口
- `GET /api/help-doc/restore-history` - 获取恢复历史
- `POST /api/help-doc/restore` - 恢复数据
- `POST /api/help-doc/restore-history/:id/log` - 获取恢复日志

### 6.7 设置接口
- `GET /api/help-doc/settings/backup` - 获取备份设置
- `PUT /api/help-doc/settings/backup` - 更新备份设置

## 7. 系统配置

### 7.1 环境配置
- `VITE_API_BASE_URL` - API接口基础URL
- `VITE_MOCK_ENABLED` - 是否启用Mock数据
- `VITE_APP_TITLE` - 应用标题

### 7.2 备份配置
- `backup.autoBackup` - 是否启用自动备份
- `backup.backupInterval` - 备份间隔（daily, weekly, monthly）
- `backup.backupTime` - 备份时间
- `backup.backupRetention` - 备份保留天数
- `backup.backupPath` - 备份路径

## 8. 常见问题

### 8.1 如何创建多级分类？
在分类管理页面，创建分类时选择父分类即可创建子分类。

### 8.2 如何恢复到之前的文档版本？
在文档列表中点击文档右侧的"版本历史"按钮，在版本历史页面中找到需要恢复的版本，点击"恢复"按钮即可。

### 8.3 如何设置文档的访问权限？
在文档编辑页面的"权限控制"部分，可以设置文档的可见性（公开、私有、受保护）和允许评论。

### 8.4 如何批量操作文档？
在文档列表页面，勾选需要操作的文档，然后点击页面顶部的"批量操作"按钮，选择需要执行的操作（删除、发布、归档）。

### 8.5 如何导出文档？
在文档列表页面，勾选需要导出的文档，然后点击页面顶部的"导出"按钮，选择导出格式（Markdown、HTML、PDF）。
